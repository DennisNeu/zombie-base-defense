# zombie-base-defense
A game made in pygame

git clone the repo with: 
```console
git clone https://github.com/DennisNeu/zombie-base-defense.git
```

Move into the project directory:
```console
cd zombie-base-defense
```

make sure pygame is installed:
```console
pip install pygame
```

To start the game:
```console
python game.py
```
